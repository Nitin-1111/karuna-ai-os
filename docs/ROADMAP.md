# Roadmap

Karuna AI OS will be implemented incrementally. Each phase should add only the functionality explicitly assigned to that phase.

## Phase 1: Project Foundation

Status: Complete in this repository state.

Scope:

- Folder structure.
- Python project metadata.
- Dependency management.
- Development scripts.
- Environment template.
- Documentation skeleton.
- Empty package initialization files.

## Later Phases

Planned areas of work:

- FastAPI backend.
- LangGraph orchestration.
- Provider-agnostic LLM layer.
- Groq provider support.
- Google Gemini provider support.
- Specialist AI agents.
- Local document storage.
- API surface.
- Testing.
- Documentation expansion.

Implementation details for later phases are intentionally not documented here.
